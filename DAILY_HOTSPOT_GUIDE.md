# 每日热点模块使用指南

## 概述
每日热点模块已成功添加到你的技术博客中，位于首页的"最新文章"部分之前。该模块展示每日IT热点新闻5条，采用与最新文章相似的格式，但增加了排名和实时性显示。

## 模块特点

### 1. 视觉设计
- **排名显示**：每条热点新闻都有醒目的排名数字（1-5）
- **颜色编码**：不同排名使用不同颜色，便于区分
- **日期标记**：自动显示当前日期
- **响应式设计**：适配各种屏幕尺寸
- **暗色主题**：支持主题切换

### 2. 交互功能
- **悬停效果**：鼠标悬停时有微妙的动画效果
- **平滑滚动**：点击导航可以平滑滚动到热点区域
- **键盘快捷键**：
  - `Alt + T`：切换主题
  - `Alt + H`：跳转到每日热点区域

### 3. 自动功能
- **日期自动更新**：每天自动更新日期显示
- **动画效果**：页面加载时有渐入动画

## 如何更新热点新闻

### 方法一：手动编辑（推荐）
编辑 `index.html` 文件，找到以下部分：

```html
<section id="daily-hotspots" class="hotspots">
    <!-- 热点内容 -->
</section>
```

在 `hotspot-grid` 中更新热点卡片：

```html
<article class="hotspot-card">
    <div class="hotspot-rank">1</div>  <!-- 排名数字 -->
    <div class="hotspot-content">
        <h3><a href="新闻链接" target="_blank">新闻标题</a></h3>
        <p>新闻摘要内容...</p>
        <div class="hotspot-meta">
            <span class="source">来源：媒体名称</span>
            <span class="time">发布时间</span>
        </div>
    </div>
</article>
```

### 方法二：使用模板
可以使用以下模板快速添加新热点：

```html
<article class="hotspot-card">
    <div class="hotspot-rank">排名</div>
    <div class="hotspot-content">
        <h3><a href="链接" target="_blank">标题</a></h3>
        <p>摘要内容，建议在100-150字左右，简要介绍新闻要点。</p>
        <div class="hotspot-meta">
            <span class="source">来源：媒体</span>
            <span class="time">时间</span>
        </div>
    </div>
</article>
```

## 热点新闻来源建议

### 推荐IT新闻源
1. **TechCrunch** - 国际科技新闻
2. **The Verge** - 科技产品评测
3. **Hacker News** - 开发者社区热点
4. **GitHub Trending** - 开源项目趋势
5. **Medium** - 技术深度文章
6. **InfoQ** - 企业级技术新闻
7. **CSDN** - 国内开发者社区
8. **掘金** - 国内技术文章平台

### 内容建议
- **时效性**：选择24小时内的热点新闻
- **相关性**：优先选择与博客主题相关的技术新闻
- **多样性**：涵盖不同领域（AI、Web开发、开源、产品等）
- **质量**：选择有深度、有价值的新闻

## 自定义样式

### 修改颜色
在 `css/style.css` 中修改CSS变量：

```css
:root {
    --hotspot-rank-1: #ff6b6b;  /* 第1名颜色 */
    --hotspot-rank-2: #ffa94d;  /* 第2名颜色 */
    --hotspot-rank-3: #ffd93d;  /* 第3名颜色 */
    --hotspot-rank-4: #6bcf7f;  /* 第4名颜色 */
    --hotspot-rank-5: #4d96ff;  /* 第5名颜色 */
}
```

### 修改动画
在 `js/script.js` 中调整动画参数：

```javascript
// 修改动画延迟
hotspotCards.forEach((card, index) => {
    card.style.animationDelay = `${index * 0.15}s`;  // 调整延迟时间
});
```

## 故障排除

### 常见问题
1. **日期不更新**：检查JavaScript是否正常加载
2. **样式错乱**：检查CSS文件路径是否正确
3. **动画不生效**：检查浏览器是否支持CSS动画

### 调试方法
1. 打开浏览器开发者工具（F12）
2. 检查控制台是否有错误信息
3. 检查元素样式是否正确应用
